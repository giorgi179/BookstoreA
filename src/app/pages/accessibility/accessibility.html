Accessibility Statement

Last updated: June 2026

Our Commitment

We are committed to ensuring digital accessibility for all readers, including those with disabilities. We continually work to improve the user experience for everyone and apply relevant accessibility standards so that our stories and storefront remain open to all.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) define requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. This site aims to conform to WCAG 2.1 level AA.

Accessibility Features

  • Semantic HTML structure for screen reader compatibility
  • Keyboard navigation support across interactive elements
  • Sufficient color contrast between text and background
  • Resizable text that does not break the layout
  • Descriptive alt text for meaningful images
  • Visible focus indicators on interactive elements

Known Limitations

Despite our best efforts, some content may not yet be fully accessible. This may include third-party content, older book cover images without descriptive text, or decorative elements that have not yet been reviewed. We are actively working to address these gaps.
